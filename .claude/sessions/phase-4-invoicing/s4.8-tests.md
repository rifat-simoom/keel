## Your task
Write the complete backend integration test suite for Phase 4: (1) State machine tests: all valid transitions, all invalid transitions raise `InvalidStateTransitionError`, SENT invoice cannot be edited, DRAFT invoice can be deleted (soft), SENT+ cannot be deleted. (5) `invoice.paid` event payload includes all required fields (invoice_id, company_id, amount, currency).
