8 — Payroll (Post-MVP)

## Your task
Write the complete payroll test suite: (1) PAYE calculator tests: £12,570 → zero tax/NIC; £30,000 → correct basic rate income tax and NIC; £60,000 → correct higher rate calculation; employer NIC correctly excluded from net pay; monthly and annual calculations are consistent. (2) RTI tests (mock HMRC API): FPS XML contains required fields (NI number, tax code, gross, PAYE, NIC); successful submission sets `submitted_to_hmrc=True`; HMRC error leaves it False; second submission raises `PayrollAlreadySubmittedError`. (3) API tests (testcontainers): create employee, create payroll run (auto-calculates), approve, submit; submitted run returns 409 on second submit; non-owner cannot approve or submit (403); soft delete of employee doesn't hard-delete (FR-22); cross-tenant isolation — company B cannot see company A's employees (FP-3). (4) FPS XML validation: parse generated XML and assert all HMRC-required fields are present.

## Constraints
- All monetary assertions use Decimal (FP-2)
- HMRC API mocked with `respx` — no real HMRC calls in tests
- Cross-tenant test required (FP-3)
- Submitted-run immutability test: attempt to add an employee to a submitted run → 409 (FP-6)
- Soft delete test: deleted employee not in list query but still in DB (FR-22)
- Coverage: 80%+ on calculator, rti, and router

## Done when
